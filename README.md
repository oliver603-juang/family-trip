# Family Trip — 親子旅遊行程 App

> 當前行程：**2026 台南親子遊** (4/30 ~ 5/2)

三天兩夜台南親子自駕旅遊行程 PWA App。

## 行程概覽

| 日期 | 主題 | 重點景點 |
|------|------|---------|
| 4/30（三） | 南下探索 | 自行車文化探索館 → 隆田園區 → HARD OFF × 2 → 福爾摩沙遊艇酒店 |
| 5/1（四） | 安平漫遊 | 德陽艦園區 → HARD OFF 頂美店 → 北返 |
| 5/2（五） | 知識之旅 | 冷研碳索館 → 國立自然科學博物館 → 約客厚禮手創館 |

## 功能

- **行程管理** — 出發時間、停留時間、交通模式即時調整
- **記帳系統** — 手動輸入 + AI 收據辨識（拍照自動辨帳）
- **AI 防雷** — Gemini AI 即時查詢景點營業狀態、天氣、路況
- **願望清單** — 臨時想去的景點，一鍵查詢即時資訊 + 導航
- **花費統計** — 每日明細 + Email 報表發送
- **PWA 離線** — 安裝到手機桌面，離線也能用

## 使用方式

1. 開啟 GitHub Pages 網址
2. 首次使用：點右上角 ⚙️ 設定 Gemini API Key
3. 手機上可「加到主畫面」作為 PWA App 使用

## 技術棧

React 18 (CDN) · Tailwind CSS · Google Gemini API · EmailJS · PWA

## 更新行程（每次旅行前）

1. 在 Google Maps 上規劃行程
2. 圖層命名規範：`YYYYMMDD` = 日期行程、`pocket_list` = 願望清單
3. 匯出 KMZ → 上傳到 Claude 對話
4. Claude 生成新的 `config.js` → 替換 repo 中的檔案
5. 推送後手機重新整理即可（行程切換時會自動提示清除舊資料）
